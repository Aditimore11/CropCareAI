# 🌱 CropCareAI

## Overview

CropCareAI is an AI-powered smart agriculture platform designed to help farmers make informed decisions through crop recommendation and disease prediction. The system leverages Machine Learning techniques to analyze agricultural data and provide accurate insights for improving crop productivity and sustainability.

---

## Features

* 🌾 Crop Recommendation System
* 🦠 Crop Disease Detection
* 📊 Data Analytics and Visualization
* 🤖 Machine Learning-Based Predictions
* 📈 User-Friendly Dashboard
* 🌍 Smart Farming Support

## Installation

### Clone the Repository

```bash
git clone https://github.com/Aditimore11/CropCareAI.git
cd CropCareAI
```

### Install Node.js Dependencies

```bash
npm install
```

### Install Python Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `config.env` file and add:

```env
GEMINI_API_KEY=your_api_key
MONGODB_URI=your_database_uri
```

### Run the Application

```bash
npm start
```
